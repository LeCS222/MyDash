# MyDash

Персональная стартовая страница из виджетов: часы, погода, заметки и другие блоки, которые вы добавите сами.

## Запуск

Приложение использует ES-модули и fetch для конфига — нужен локальный сервер:

```bash
cd mydash
npx --yes serve -p 3000
```

Или через Python:

```bash
cd mydash
python -m http.server 3000
```

Откройте в браузере: http://localhost:3000

## Структура проекта

```
mydash/
├── index.html              # Разметка
├── styles/main.css         # Стили и темы
├── js/
│   ├── app.js              # Точка входа
│   ├── storage.js          # Обёртка localStorage
│   ├── registry.js         # Реестр виджетов
│   └── widgets/            # Виджеты
├── data/default-config.json
└── .cursor/                # Правила и скиллы Cursor
```

## Контракт виджета

Каждый виджет — ES-модуль с полями `id`, `title`, `init()`, `render(container)`.

Подробности — в `.cursor/rules/widgets.mdc` и скилле `add-widget`.

## Задание

Расширьте приложение: добавьте новые виджеты, кастомизацию раскладки и исправьте ошибки. Подробное задание — в документе `MyDash_ТЗ_практическое_задание.md`.

## Cursor

- **Правила:** `.cursor/rules/` — контракт виджетов, стек, формат конфигов
- **Скилл:** `/add-widget` — пошаговое добавление нового виджета
